CodeProject.AI Server Setup Guide

Before you begin: You need to have CodeProject.AI Server installed and running on your network.

Step 1: Install CodeProject.AI Server

Docker Installation (Recommended)

docker run -d --name codeproject-ai-server \
  -p 8000:32168 \
  -v codeproject-ai:/app/storage \
  --restart unless-stopped \
  codeproject/codeproject-ai-server:latest

Direct Installation

Download from GitHub and follow the installation instructions.

Step 2: Configure CodeProject.AI Server

Required Modules

  • Face Detection: Automatically installed with the server
  • Face Recognition: Automatically installed with the server

Access the Web Interface

Open your browser and navigate to: http://your-server-ip:8000

Verify that the Face Detection and Face Recognition modules are running.

Step 3: Configure FND Connection

Server Settings

  • Server Host: IP address or hostname of your CodeProject.AI server
  • Server Port: Default is 8000 (or the port you configured)
  • Use SSL: Enable if using HTTPS
  • Connection Timeout: 30 seconds is usually sufficient

Test Connection

Use the "Test Connection" button in the Server Settings tab to verify connectivity.

Step 4: Add Known Faces

Face Requirements

  • Clear, front-facing photos
  • Good lighting conditions
  • Face should be clearly visible
  • Recommended size: 200x200 pixels minimum

Adding Faces

  1. Go to the "Face Management" tab
  2. Click "Add New Face"
  3. Enter the person's first and last name
  4. Upload a clear photo of their face
  5. Save the face record

Step 5: Enable and Test

Final Steps

  1. Enable the CodeProject.AI service using the toggle at the top
  2. Configure which features to enable (detection/recognition)
  3. Test with a Frigate event containing a person
  4. Check notifications for facial recognition results

Troubleshooting

  • Connection failed: Check server IP, port, and firewall settings
  • No faces detected: Verify face detection is enabled in CodeProject.AI
  • No recognition: Ensure you have added known faces to the database
  • Poor accuracy: Use higher quality face images with better lighting
Important: CodeProject.AI Server requires significant computational resources. Ensure your server has adequate CPU and RAM for optimal performance.
Need help? Visit the CodeProject.AI documentation for detailed setup instructions.

Pending Faces Auto-Processing

Automatic Processing: When facial recognition is disabled, person events are stored for later analysis. Enable automatic processing to analyze these events periodically using CodeProject.AI.

When enabled, pending face events will be automatically processed every few hours

Interval in hours between automatic processing runs (1-168 hours)

Note: Automatic processing requires both facial recognition service and CodeProject.AI to be enabled and properly configured.

How it works

  1. When facial recognition is disabled, person events are automatically stored
  2. Enable automatic processing in this section
  3. Set the desired processing interval (e.g., every 6 hours)
  4. The system will automatically process all pending events at the specified interval
  5. Results are stored with the original events for review
